# 🎯 SpigotMC Account Scanner

Tool để quét các tài khoản SpigotMC và kiểm tra các plugin đã mua (paid plugins).

## ✨ Features

- 🔍 Quét multiple SpigotMC accounts
- 💰 Phát hiện plugins trả phí
- 📊 Báo cáo chi tiết với thống kê
- 💾 Lưu kết quả tự động
- 🎯 Dễ sử dụng với CLI
